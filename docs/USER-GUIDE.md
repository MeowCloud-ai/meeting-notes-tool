# USER-GUIDE.md — MeowMeet 使用手冊

## 快速開始

### 1. 安裝
👉 見 [INSTALL-GUIDE.md](INSTALL-GUIDE.md)

### 2. 開會前準備
1. 確認「系統設定」→「聲音」→「輸出」設為**多重輸出裝置**
2. 開啟 MeowMeet

### 3. 開始錄製
1. 開啟 Google Meet 會議
2. 在 MeowMeet 點 **🎙️ Start Recording**
3. 正常開會，MeowMeet 在背景自動錄製

### 4. 結束錄製
1. 會議結束後，點 **⏹ Stop**
2. 等待 3-5 分鐘（處理最後一段音訊 + 生成摘要）
3. 收到：
   - 📄 Google Docs 完整會議紀錄
   - 📧 Email 摘要通知

## 功能說明

### 🎙️ 錄製控制

| 按鈕 | 功能 |
|------|------|
| **Start** | 開始錄製，音訊每 3 分鐘自動切段處理 |
| **Pause** | 暫停錄製（休息時間用） |
| **Resume** | 繼續錄製 |
| **Stop** | 結束錄製，開始生成會議紀錄 |

### 📊 狀態顯示

- **錄音時長**: 目前已錄製的總時間
- **音量指示器**: 即時顯示收音狀況（確認有在錄到）
- **轉錄進度**: 已完成轉錄的段數 / 總段數

### 📄 會議紀錄內容

自動產出的 Google Docs 包含：

```
📋 會議資訊
├── 日期：2026-02-18
├── 時長：45 分鐘
└── 產出時間：2026-02-18 15:48

📝 會議摘要
  本次會議討論了 XXX，決定 YYY...（3-5 句）

✅ 行動項目
  - [ ] 講者 A：完成 XX 報告（截止 2/20）
  - [ ] 講者 B：確認 YY 預算

📌 決策紀錄
  1. 決定使用方案 A，因為...
  2. 時程從 3 月改到 4 月

💬 關鍵討論點
  - 關於預算分配的討論...
  - 技術方案比較...

📃 逐字稿
  [00:00] 講者 A：大家好，今天要討論...
  [00:15] 講者 B：我先報告一下進度...
```

### 📧 Email 摘要

Email 包含精簡版：
- 會議摘要
- 行動項目清單
- Google Docs 連結（點擊看完整紀錄）

## 設定選項

在 MeowMeet 的 **⚙️ Settings** 頁面：

| 選項 | 說明 | 預設值 |
|------|------|--------|
| 音訊輸入 | 選擇 BlackHole 2ch | 自動偵測 |
| 分段間隔 | 每幾分鐘切一段處理 | 3 分鐘 |
| Whisper 模型 | tiny/base/small/medium | small |
| 語言 | 轉錄語言 | 中文 (zh) |
| Docs 資料夾 | Google Docs 存放位置 | MeowMeet Notes |
| Email 收件人 | 摘要寄送對象 | （需設定） |

### 模型選擇建議

| 模型 | 大小 | 速度 | 準確度 | 建議 |
|------|------|------|--------|------|
| tiny | 75 MB | ⚡ 最快 | ⭐⭐ | 測試用 |
| base | 142 MB | ⚡ 快 | ⭐⭐⭐ | 簡短會議 |
| **small** | 466 MB | 🔄 中等 | ⭐⭐⭐⭐ | **推薦** |
| medium | 1.5 GB | 🐢 慢 | ⭐⭐⭐⭐⭐ | 需要高精確度 |

## 常見問題 (FAQ)

### Q: 錄製時我自己聽得到聲音嗎？
A: 聽得到！透過「多重輸出裝置」設定，聲音同時送到你的耳機和 MeowMeet。

### Q: 可以錄多長的會議？
A: 沒有上限。音訊每 3 分鐘切段處理，不會累積記憶體。

### Q: 會錄到電腦其他聲音嗎？
A: 會。BlackHole 擷取所有系統音訊。建議錄製時關閉其他播放音訊的應用程式。

### Q: 錄音檔會上傳嗎？
A: 不會。音訊處理全部在你的電腦本機完成。只有最終的文字摘要會送到 Gemini API 和 Google Docs。

### Q: 如果轉錄結果不準確？
A: 嘗試換用更大的 Whisper 模型（Settings → Whisper Model → medium）。

### Q: 忘記按停止怎麼辦？
A: MeowMeet 會持續錄製。回來按停止就好，之前的都不會遺失。

## 故障排除

| 症狀 | 可能原因 | 解決方法 |
|------|----------|----------|
| 音量指示器沒反應 | 系統輸出未設為多重輸出裝置 | 見安裝手冊「設定多重輸出裝置」 |
| 轉錄是亂碼 | 語言設定錯誤 | Settings → 語言 → 中文 |
| 轉錄很慢 | 模型太大 or CPU 不夠 | 換用 small 模型 |
| Google Docs 沒建立 | OAuth 過期 | Settings → 重新連接 Google |
| Email 沒收到 | 收件人設定錯誤 | 檢查 Settings → Email 收件人 |
| 應用程式當掉 | 記憶體不足 | 關閉其他應用程式，或用較小的模型 |
