# 使用指南

## 快速開始（3 步驟）

1. **安裝** — 從 Chrome Web Store 安裝 MeowMeet
2. **登入** — 使用 Google 帳號登入
3. **錄音** — 在任何會議頁面點擊 MeowMeet 圖示，按下錄音按鈕

## 功能說明

### 🎤 錄音

- 點擊 Extension 圖示開啟面板
- 按下紅色按鈕開始錄音
- 支援暫停/繼續
- 錄音完成後自動上傳

### 📝 逐字稿

- 錄音結束後自動產生逐字稿
- 支援多語言（中文/英文）
- 可在錄音詳情中查看

### 🤖 AI 摘要

- 自動產出會議重點
- 提取待辦事項與負責人
- 標記關鍵對話

### 📴 離線模式

- 斷網時錄音會暫存於本地
- 恢復網路後自動同步上傳

## 免費方案

- 每月 3 場免費錄音
- 每場最長 60 分鐘
- 升級到 Starter 方案享受更多額度

## FAQ

**Q: 支援哪些會議平台？**
A: 任何在 Chrome 瀏覽器中的會議都支援，包括 Google Meet、Zoom Web、Microsoft Teams Web 等。

**Q: 錄音資料安全嗎？**
A: 所有資料透過加密傳輸，儲存於安全的雲端服務。詳見[隱私政策](privacy-policy.html)。

**Q: 如何刪除我的資料？**
A: 在 Extension 中可刪除個別錄音，或聯繫 support@meowmeet.app 完整刪除帳號。
