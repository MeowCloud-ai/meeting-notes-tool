🐱 MeowMeet 隱私政策

最後更新：2026-02-21

我們收集什麼資料

  • Google 帳號資訊：email、名稱、頭像（用於登入與顯示）
  • 會議錄音：透過瀏覽器分頁錄製的音訊（用於轉錄與摘要）
  • 轉錄文字與摘要：由 AI 產生的會議內容
  • 使用統計：錄音次數（用於方案限制）

我們如何使用資料

  • 提供會議錄音、轉錄、摘要服務
  • 管理用戶帳號與方案限制
  • 改善服務品質

資料儲存

  • 音訊檔案與資料儲存於 Supabase（AWS 基礎設施）
  • 轉錄使用 Deepgram API
  • 摘要使用 Google Gemini API

資料分享

我們不會出售您的資料。僅在以下情況分享：

  • 第三方 API 處理（Deepgram、Google Gemini）——僅限提供服務所需
  • 法律要求

如何刪除資料

  • 在 Extension 中刪除個別錄音
  • 聯繫 support@meowmeet.app 要求完整刪除帳號與所有資料

聯絡我們

如有任何隱私相關問題，請聯繫：support@meowmeet.app
