# PRD — MeowMeet 產品需求規格

## 產品願景
一鍵開始，會後 3 分鐘收到完整會議紀錄。

## 目標用戶
- 經常使用 Google Meet 開會的工作者
- 需要會議紀錄但沒時間手寫的人

## 核心功能

### F1: 音訊擷取
- 透過 BlackHole 擷取 Google Meet 音訊
- 每 3-5 分鐘自動切段
- 顯示錄音狀態（時長、音量指示）
- 支援暫停 / 繼續

### F2: 語音轉錄
- 使用本機 Whisper 模型（small/medium）
- 中文為主要語言
- 即時處理每一段音訊（不等到會後）
- 轉錄完成後顯示進度

### F3: 講者辨識（Speaker Diarization）
- 至少分辨講者 A、講者 B...
- 跨段落保持講者一致性
- 可選：會後手動標註講者姓名

### F4: AI 會議摘要
- 會議結束後，用 Gemini 生成：
  - 會議摘要（3-5 句）
  - 行動項目清單（誰、做什麼、何時）
  - 決策紀錄
  - 關鍵討論點
- 逐字稿帶講者標註

### F5: Google Docs 輸出
- 自動建立 Google Docs
- 標準格式模板
- 歸檔到指定 Google Drive 資料夾
- 回傳文件連結

### F6: Email 通知
- 會議結束後寄送摘要 Email
- 包含：摘要 + 行動項目 + Docs 連結
- 收件人可設定

## 非功能需求

### 效能
- 轉錄延遲 < 30 秒/段
- 會後產出時間 < 5 分鐘
- 記憶體使用 < 2GB

### 安全
- OAuth token 安全存儲（Electron safeStorage）
- 音訊檔案僅存本機，處理完可自動刪除
- 不上傳原始音訊到任何雲端

### 使用者體驗
- 一鍵開始/停止
- 系統匣常駐
- 首次使用引導（安裝 BlackHole + Google 授權）
